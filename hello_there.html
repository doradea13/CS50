To add a file in the repository:
git add $file$
To add a commitment
git commit -m "Your message of changes here"
Getting the git status you can see how often commitments have you done so far:
git status
Adding the comand to push, you upload the data you have done:
git push -u
The commit -m is just to commit a unique file at a time, if u want to commit a lot of files, use:
git commit -am "message here"
